Introduction to HTML

What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used for creating and designing webpages. HTML elements are the fundamental building blocks of web pages, providing structure and meaning to the content.

HTML documents are composed of elements such as headings, paragraphs, lists, and links. Each HTML element is defined by tags and attributes that help format and organize the content on a webpage.

Basic Structure of an HTML Document

An HTML document follows a basic structure which consists of the following elements:

  • <!DOCTYPE html> - Declares the document type and HTML version.
  • <html> - The root element that wraps the entire HTML document.
  • <head> - Contains meta-information about the document, including the title, character encoding, and links to stylesheets.
  • <body> - Contains the visible content of the document, such as headings, paragraphs, images, and links.

Example of a Simple HTML Document

Here is an example of a basic HTML document:


<!DOCTYPE html>
<html>
    <head>
        <title>My First HTML Page</title>
    </head>
    <body>
        <h1>Welcome to My Website</h1>
        <p>This is a paragraph of text.</p>
    </body>
</html>

HTML Tags and Elements

HTML tags are used to define elements on a webpage. Tags typically come in pairs: an opening tag and a closing tag. The opening tag defines the beginning of an element, while the closing tag defines its end.

For example:


<tagname>Content</tagname>

Common HTML tags include:

  • <h1> to <h6> - Headings, with <h1> being the largest and <h6> the smallest.
  • <p> - Defines a paragraph.
  • <a> - Defines a hyperlink. Use the href attribute to specify the URL.
  • <img> - Embeds an image. Use the src attribute to specify the image source.
  • <div> - Defines a division or section. Useful for grouping content.
  • <span> - Defines a section in a document, usually for styling purposes.

HTML Attributes

Attributes provide additional information about HTML elements. They are added to the opening tag of an element and are written in name/value pairs. For example:


<elementname attribute="value">Content</elementname>

Common attributes include:

  • href - Specifies the URL for hyperlinks.
  • src - Specifies the URL of an image.
  • alt - Provides alternative text for images.
  • class - Assigns one or more class names to an element for styling with CSS.
  • id - Provides a unique identifier for an element, used for styling with CSS or accessing with JavaScript.
  • style - Applies inline CSS styling directly to an element.

HTML Document Metadata

Metadata is information about the HTML document that is not displayed on the webpage but is used by browsers and search engines. Metadata is placed within the <head> section of the document.

  • <meta charset="UTF-8"> - Specifies the character encoding for the document, ensuring text is displayed correctly.
  • <meta name="description" content="A description of the page"> - Provides a brief description of the page's content for search engines.
  • <meta name="keywords" content="HTML, web development, tutorial"> - Lists keywords relevant to the page.
  • <link rel="stylesheet" href="styles.css"> - Links to an external CSS stylesheet for styling the document.
  • <title> - Defines the title of the document, displayed in the browser's title bar or tab.

HTML Forms

Forms are used to collect user input. They can contain various elements such as text fields, checkboxes, radio buttons, and submit buttons.

Here is a basic example of an HTML form:


	<form action="/submit" method="post">
    <label for="name">Name:</label>
		<input type="text" id="name" name="name">
    <br>
    <label for="email">Email:</label>
		<input type="email" id="email" name="email">
    <br>
		<input type="submit" value="Submit">
</form>

Form elements include:

  • <input> - Defines an input field. Use the type attribute to specify the type of input (text, email, password, etc.).
  • <textarea> - Defines a multi-line text input field.
  • <select> - Defines a drop-down list. Use <option> to define options within the list.
  • <button> - Defines a clickable button.
  • <fieldset> - Groups related form elements.
  • <legend> - Defines a caption for a <fieldset>.

HTML Tables

HTML tables are used to display tabular data. A table is defined with the <table> tag, and it contains rows defined by the <tr> tag. Each row contains cells defined by <td> (table data) and <th> (table header) tags.

Here is an example of a simple HTML table:


<table border="1">
    <thead>
        <tr>
            <th>Header 1</th>
            <th>Header 2</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Data 1</td>
            <td>Data 2</td>
        </tr>
    </tbody>
</table>

Table tags include:

  • <thead> - Groups the header content in a table.
  • <tbody> - Groups the body content in a table.
  • <tfoot> - Groups the footer content in a table.

HTML Semantic Elements

Semantic elements are HTML5 elements that carry meaning about the content they enclose, providing a clear structure to the webpage. These elements help with accessibility and search engine optimization.

Examples of semantic elements include:

  • <header> - Defines a header for a document or section.
  • <nav> - Defines a navigation bar.
  • <article> - Defines an independent piece of content.
  • <section> - Defines a section in a document.
  • <aside> - Defines content aside from the main content.
  • <footer> - Defines a footer for a document or section.
  • <main> - Defines the main content of the document.
  • <figure> - Defines self-contained content, such as images or diagrams.
  • <figcaption> - Provides a caption for a <figure> element.

HTML Multimedia Elements

HTML supports embedding multimedia content such as images, audio, and video. These elements enhance the visual and interactive aspects of a webpage.

Examples include:


<img src="image.jpg" alt="Description of image">
<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
    Your browser does not support the audio element.
</audio>
<video width="320" height="240" controls>
    <source src="video.mp4" type="video/mp4">
    Your browser does not support the video tag.
</video>

Attributes for these elements include:

  • controls - Adds playback controls for audio and video elements.
  • autoplay - Specifies that the audio or video should start playing automatically.
  • loop - Specifies that the audio or video should loop.
  • muted - Specifies that the audio should be muted by default.

HTML Document Validation

Validating your HTML document ensures that it adheres to the standards set by the World Wide Web Consortium (W3C). This helps in maintaining compatibility across different browsers and improving the overall quality of your webpage.

You can validate your HTML document using online tools such as the W3C Markup Validation Service.

HTML Best Practices

Following best practices when writing HTML helps ensure that your code is clean, maintainable, and accessible. Some best practices include:

  • Use semantic elements to give meaning to your content.
  • Ensure that your HTML is properly nested and structured.
  • Provide alternative text for images and multimedia elements.
  • Use comments to explain sections of your code.
  • Validate your HTML to catch errors and improve compatibility.
  • Follow a consistent coding style for readability.
