Data Types and Variables

In this section, we will explore the different data types available in Python and learn how to declare and use variables.

What are Data Types?

Data types are classifications of data that tell the interpreter how the programmer intends to use the data. Common data types in Python include:

  • Integers: Whole numbers, e.g., 1, 2, 3.
  • Floats: Decimal numbers, e.g., 1.5, 2.7, 3.14.
  • Strings: Sequence of characters, e.g., "Hello, World!"
  • Booleans: True or False values.
  • Lists: Ordered collection of items, e.g., [1, 2, 3].
  • Tuples: Immutable ordered collection of items, e.g., (1, 2, 3).
  • Dictionaries: Key-value pairs, e.g., {'name': 'John', 'age': 30}.

What are Variables?

Variables are containers for storing data values. Unlike other programming languages, Python has no command for declaring a variable. A variable is created the moment you first assign a value to it.

Example:


x = 5
y = "Hello, World!"

In the example above, x is a variable with the value of 5, and y is a variable with the value of "Hello, World!"

Data Type Examples

Integers and Floats


x = 10       # integer
y = 2.5      # float
print(x)
print(y)

Strings


name = "Alice"
print(name)

Booleans


is_student = True
is_teacher = False
print(is_student)
print(is_teacher)

Lists


numbers = [1, 2, 3, 4, 5]
print(numbers)

Tuples


coordinates = (10.0, 20.0)
print(coordinates)

Dictionaries


person = {
    "name": "John",
    "age": 30,
    "city": "New York"
}
print(person)

Variable Naming Rules

When naming variables in Python, follow these rules:

  • Variable names must start with a letter or the underscore character.
  • Variable names cannot start with a number.
  • Variable names can only contain alpha-numeric characters and underscores (A-z, 0-9, and _ ).
  • Variable names are case-sensitive (age, Age, and AGE are three different variables).

Type Conversion

Python allows you to convert from one data type to another. This is called type conversion.

Example:


x = 1       # int
y = 2.8     # float
z = "3"     # str

# Convert from int to float
a = float(x)

# Convert from float to int
b = int(y)

# Convert from str to int
c = int(z)

print(a)
print(b)
print(c)
