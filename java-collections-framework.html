Java Collections Framework

Java Collections Framework

The Java Collections Framework provides a set of interfaces and classes for storing and manipulating groups of data. It includes structures like lists, sets, and maps, and supports various algorithms for processing collections.

1. Introduction to Collections

The Collections Framework is a unified architecture for representing and manipulating collections. It includes a set of interfaces, implementations, and algorithms that provide various functionalities for managing data.

Interfaces

  • Collection: The root interface in the collection hierarchy. It is the most general form of a collection and includes most of the operations that can be performed on collections.
  • List: An ordered collection that allows duplicate elements. Common implementations include `ArrayList` and `LinkedList`.
  • Set: A collection that does not allow duplicate elements. Common implementations include `HashSet` and `TreeSet`.
  • Map: A collection that maps keys to values. It does not allow duplicate keys. Common implementations include `HashMap` and `TreeMap`.

Implementations

  • ArrayList: Implements the `List` interface. It provides a resizable array implementation and is generally preferred for most applications due to its efficiency in accessing elements.
  • LinkedList: Implements the `List` and `Deque` interfaces. It provides a doubly-linked list implementation and is more efficient for inserting and removing elements compared to `ArrayList`.
  • HashSet: Implements the `Set` interface. It uses a hash table for storage and provides constant-time performance for basic operations.
  • TreeSet: Implements the `Set` interface. It uses a red-black tree and maintains elements in a sorted order.
  • HashMap: Implements the `Map` interface. It uses a hash table and provides constant-time performance for basic operations.
  • TreeMap: Implements the `Map` interface. It uses a red-black tree and maintains keys in a sorted order.

2. Working with Lists

The `List` interface represents an ordered collection of elements. Here's how you can work with `ArrayList`:

import java.util.ArrayList;
import java.util.List;

public class ListExample {
    public static void main(String[] args) {
        List list = new ArrayList<>();
        list.add("Java");
        list.add("Python");
        list.add("C++");

        System.out.println("List: " + list);
        System.out.println("First Element: " + list.get(0));

        list.remove("Python");
        System.out.println("Updated List: " + list);
    }
}

3. Working with Sets

The `Set` interface represents a collection that does not allow duplicate elements. Here's how you can work with `HashSet`:

import java.util.HashSet;
import java.util.Set;

public class SetExample {
    public static void main(String[] args) {
        Set set = new HashSet<>();
        set.add("Java");
        set.add("Python");
        set.add("C++");
        set.add("Java"); // Duplicate element

        System.out.println("Set: " + set);
    }
}

4. Working with Maps

The `Map` interface represents a collection of key-value pairs. Here's how you can work with `HashMap`:

import java.util.HashMap;
import java.util.Map;

public class MapExample {
    public static void main(String[] args) {
        Map map = new HashMap<>();
        map.put("Java", "Programming Language");
        map.put("Python", "Programming Language");
        map.put("C++", "Programming Language");

        System.out.println("Map: " + map);
        System.out.println("Value for 'Java': " + map.get("Java"));

        map.remove("Python");
        System.out.println("Updated Map: " + map);
    }
}

5. Collections Utility Methods

The `Collections` class provides static utility methods for various operations on collections:

import java.util.ArrayList;
import java.util.Collections;
import java.util.List;

public class CollectionsUtilityExample {
    public static void main(String[] args) {
        List list = new ArrayList<>();
        list.add("Java");
        list.add("Python");
        list.add("C++");

        Collections.sort(list);
        System.out.println("Sorted List: " + list);
    }
}
