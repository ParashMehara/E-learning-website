Pointers and Memory Management

Pointers and memory management are crucial aspects of C programming. They allow you to directly access and manipulate memory, providing powerful capabilities for managing resources and optimizing performance. This section covers the basics of pointers and memory management.

Pointers

Pointers are variables that store the address of another variable. They provide a way to directly access and modify the memory location of variables.

Pointer Declaration

data_type *pointer_name;

Pointer Initialization

pointer_name = &variable_name;

Example

#include <stdio.h>

int main() {
    int num = 10;
    int *ptr = #

    printf("Value of num: %d\n", num);
    printf("Address of num: %p\n", &num);
    printf("Value of ptr: %p\n", ptr);
    printf("Value pointed to by ptr: %d\n", *ptr);

    return 0;
}

Memory Management

In C, memory management is done manually using functions to allocate and deallocate memory dynamically. This allows you to handle memory efficiently and avoid memory leaks.

Dynamic Memory Allocation

Use the following functions to allocate and deallocate memory:

  • malloc: Allocates a block of memory.
  • calloc: Allocates memory for an array and initializes it to zero.
  • realloc: Resizes a previously allocated block of memory.
  • free: Deallocates a previously allocated block of memory.

Example

#include <stdio.h>
#include <stdlib.h>

int main() {
    int *arr;
    int n = 5;

    // Allocate memory
    arr = (int *)malloc(n * sizeof(int));
    if (arr == NULL) {
        printf("Memory allocation failed\n");
        return 1;
    }

    // Initialize and print array elements
    for (int i = 0; i < n; i++) {
        arr[i] = i * 10;
        printf("Element %d: %d\n", i, arr[i]);
    }

    // Deallocate memory
    free(arr);

    return 0;
}

Pointer to Pointer

Pointers can also point to other pointers. This allows for more complex data structures and memory management techniques.

Example

#include <stdio.h>

int main() {
    int num = 20;
    int *ptr1 = #
    int **ptr2 = &ptr1;

    printf("Value of num: %d\n", num);
    printf("Value of ptr1: %p\n", ptr1);
    printf("Value pointed to by ptr1: %d\n", *ptr1);
    printf("Value of ptr2: %p\n", ptr2);
    printf("Value pointed to by ptr2: %p\n", *ptr2);
    printf("Value pointed to by *ptr2: %d\n", **ptr2);

    return 0;
}
