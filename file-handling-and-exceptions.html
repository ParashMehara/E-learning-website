File Handling and Exceptions

File handling is an important part of any web application. Python has several functions for creating, reading, updating, and deleting files. Exception handling is crucial for writing robust code that can handle errors gracefully.

Opening and Closing Files

You can open a file using the open() function. The close() method is used to close a file.


# Opening a file
file = open('example.txt', 'r')

# Closing a file
file.close()

Reading from Files

To read from a file, you can use methods like read(), readline(), or readlines().


file = open('example.txt', 'r')
content = file.read()
print(content)
file.close()

Writing to Files

You can write to a file using the write() or writelines() methods. Ensure you open the file in write mode ('w') or append mode ('a').


file = open('example.txt', 'w')
file.write('Hello, World!')
file.close()

Exception Handling

Exceptions are errors that occur during the execution of a program. Python provides a way to handle exceptions to prevent the program from crashing.


try:
    file = open('example.txt', 'r')
    content = file.read()
    print(content)
except FileNotFoundError:
    print('The file was not found.')
finally:
    file.close()

Common Exceptions

Some common exceptions in Python include:

  • FileNotFoundError
  • ZeroDivisionError
  • ValueError
  • TypeError
  • IndexError

Try, Except, Else, Finally

You can use the try, except, else, and finally blocks to handle exceptions in a more structured way.


try:
    result = 10 / 0
except ZeroDivisionError:
    print('You cannot divide by zero.')
else:
    print('The division was successful.')
finally:
    print('This code runs no matter what.')
