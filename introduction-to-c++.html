Introduction to C++

Introduction to C++

C++ is a general-purpose programming language that supports object-oriented, procedural, and generic programming. It is widely used for developing system/software applications, game development, and high-performance computing.

History and Overview

C++ was developed by Bjarne Stroustrup as an enhancement to the C programming language. It was designed to provide better control over system resources and memory, while still offering high-level features for easier programming.

Basic Syntax

C++ shares many syntactical elements with C, including the use of semicolons to end statements and curly braces to define code blocks. Here is a basic example of a C++ program:

#include <iostream>

int main() {
    std::cout << "Hello, C++!" << std::endl;
    return 0;
}

Key Components

  • Header Files: These files contain declarations of functions and classes. For example, <iostream> is used for input and output operations.
  • Main Function: The main function is the entry point of a C++ program. It must return an integer.
  • Namespaces: C++ uses namespaces to avoid naming conflicts. The standard namespace is std, which includes common functions and objects like cout and endl.

Compiling and Running C++ Programs

To compile and run a C++ program, use a compiler like g++ for GNU or clang++ for Clang. Here is an example command for compiling a file named example.cpp:

g++ example.cpp -o example
./example

Basic Input and Output

C++ uses iostream for standard input and output. The cout object is used for output, and cin is used for input.

#include <iostream>

int main() {
    int age;
    std::cout << "Enter your age: ";
    std::cin >> age;
    std::cout << "You are " << age << " years old." << std::endl;
    return 0;
}

Conclusion

This introductory section covers the basics of C++ programming. As you advance, you'll learn about more complex topics such as object-oriented programming, templates, and the Standard Template Library (STL).
