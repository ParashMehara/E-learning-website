CSS Basics

Introduction to CSS

Cascading Style Sheets (CSS) is a stylesheet language used to describe the presentation of a document written in HTML or XML. CSS defines how HTML elements are to be displayed on screen, paper, or in other media. It enables you to control the layout, colors, fonts, and overall appearance of web pages.

CSS is essential for creating visually appealing and well-structured web pages. It separates content from design, allowing for a more flexible and maintainable approach to web development.

CSS Syntax

CSS rules are made up of two main parts: selectors and declarations. The basic syntax of a CSS rule is:

selector {
    property: value;
}
  • Selector: Specifies which HTML elements the style will apply to. Examples include element names, class names, and IDs.
  • Property: Defines the aspect of the element that you want to change (e.g., color, font-size).
  • Value: Specifies the value for the property (e.g., red, 16px).

Here’s an example of a CSS rule that changes the color of all paragraphs to blue:

p {
    color: blue;
}

CSS Selectors

Selectors are used to target the HTML elements that you want to style. Here are some common types:

  • Element Selector: Targets elements by their tag name. Example: h1, p.
  • Class Selector: Targets elements with a specific class attribute. Example: .my-class. Use this selector for multiple elements with the same style.
  • ID Selector: Targets an element with a specific ID attribute. Example: #my-id. IDs should be unique within a page.
  • Attribute Selector: Targets elements with a specific attribute. Example: [type="text"].
  • Universal Selector: Targets all elements. Example: *.

Here’s an example of using different selectors:

/* Element Selector */
p {
    color: green;
}

/* Class Selector */
.my-class {
    font-size: 20px;
}

/* ID Selector */
#my-id {
    background-color: yellow;
}

/* Attribute Selector */
input[type="text"] {
    border: 1px solid black;
}

/* Universal Selector */
* {
    box-sizing: border-box;
}

The CSS Box Model

The CSS box model describes how elements are structured and how their dimensions are calculated. It consists of the following components:

  • Content: The actual content of the box (text, images, etc.).
  • Padding: Space between the content and the border, inside the element.
  • Border: Surrounds the padding and content.
  • Margin: Space outside the border, separating the element from other elements.

Here’s an example demonstrating the box model properties:

/* Example of the box model in CSS */
.box {
    width: 300px;
    padding: 20px;
    border: 5px solid black;
    margin: 15px;
}

CSS Layout Techniques

CSS provides various techniques for creating layouts. Here are some commonly used ones:

  • Floats: Originally used to create layouts and wrap text around images. Floats are less commonly used now in favor of Flexbox and Grid.
  • Flexbox: A layout module that provides an easy way to align items and distribute space within a container. Flexbox is designed for one-dimensional layouts (row or column).
  • Grid: A layout system that allows for two-dimensional layouts with rows and columns. It is more powerful and flexible than Flexbox for complex layouts.

Examples of Flexbox and Grid in CSS:

/* Flexbox example */
.container {
    display: flex;
    justify-content: space-between;
}

.item {
    flex: 1;
    margin: 10px;
}

/* Grid example */
.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 10px;
}

.grid-item {
    background-color: lightgrey;
    padding: 20px;
}

Advanced CSS Topics

Advanced CSS features allow for more control and creativity in web design:

  • CSS Variables: Allow you to store and reuse values throughout your CSS. Example:
    • :root {
    --main-bg-color: #f4f4f4;
    --main-text-color: #333;
}

body {
    background-color: var(--main-bg-color);
    color: var(--main-text-color);
}
  • CSS Transitions: Enable smooth changes between property values. Example:
    • .box {
    width: 100px;
    height: 100px;
    background-color: red;
    transition: background-color 0.3s ease;
}

.box:hover {
    background-color: blue;
}
  • CSS Animations: Allow you to create complex animations using keyframes. Example:
    • @keyframes slide {
    from {
        transform: translateX(-100%);
    }
    to {
        transform: translateX(0);
    }
}

.slide-in {
    animation: slide 0.5s ease-in-out;
}
  • Media Queries: Allow you to apply styles based on device characteristics like screen width. Example:
    • @media (max-width: 600px) {
    .container {
        flex-direction: column;
    }
}
