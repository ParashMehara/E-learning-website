Classes and Objects

Classes and Objects

In C++, classes and objects are fundamental concepts of object-oriented programming (OOP). They allow you to define custom data types and create instances of those types.

Introduction to Classes

A class is a blueprint for creating objects. It encapsulates data and functions that operate on the data. Here is a basic example of a class definition:

#include <iostream>

class Car {
public:
    std::string brand;
    int year;

    void display() {
        std::cout << "Brand: " << brand << ", Year: " << year << std::endl;
    }
};

int main() {
    Car myCar;
    myCar.brand = "Toyota";
    myCar.year = 2020;
    myCar.display();
    return 0;
}

Class Members

  • Data Members: Variables defined inside a class that hold data.
  • Member Functions: Functions defined inside a class that operate on data members.
  • Access Specifiers: Control the accessibility of class members. The main specifiers are public, protected, and private.

Objects and Instantiation

#include <iostream>

class Person {
public:
    std::string name;
    int age;

    void introduce() {
        std::cout << "Hi, I am " << name << " and I am " << age << " years old." << std::endl;
    }
};

int main() {
    Person person1;
    person1.name = "Alice";
    person1.age = 30;
    person1.introduce();

    Person person2;
    person2.name = "Bob";
    person2.age = 25;
    person2.introduce();

    return 0;
}

Constructors and Destructors

Constructors are special member functions that are automatically called when an object is created. Destructors are called when an object is destroyed. They help in initializing and cleaning up resources, respectively.

#include <iostream>

class Rectangle {
public:
    int width, height;

    // Constructor
    Rectangle(int w, int h) : width(w), height(h) {}

    // Destructor
    ~Rectangle() {
        std::cout << "Rectangle object is being destroyed." << std::endl;
    }

    int area() {
        return width * height;
    }
};

int main() {
    Rectangle rect(10, 5);
    std::cout << "Area of rectangle: " << rect.area() << std::endl;
    return 0;
}

Inheritance and Polymorphism

Inheritance allows a class to inherit properties and behaviors from another class. Polymorphism enables one interface to be used for a general class of actions. We will explore these advanced topics in later sections.

Conclusion

Understanding classes and objects is crucial for mastering C++ and object-oriented programming. These concepts form the foundation for creating more complex programs and systems.
