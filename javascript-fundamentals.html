JavaScript Fundamentals

Introduction to JavaScript

JavaScript is a versatile programming language that allows you to create interactive and dynamic web content. It is commonly used for client-side scripting to enhance user experiences on websites. JavaScript can be used to manipulate HTML and CSS, handle events, and perform calculations.

JavaScript is an essential skill for web developers, enabling them to create complex functionality and interactivity in web applications.

JavaScript Syntax

JavaScript syntax refers to the set of rules that define a correctly structured JavaScript program. JavaScript code is written in statements, which can be simple expressions or more complex blocks.

// Single-line comment

/*
  Multi-line comment
*/

/* Example of a JavaScript statement */
let message = 'Hello, World!';
console.log(message); // Output: Hello, World!

Variables

Variables in JavaScript are used to store data values. You can declare variables using var, let, or const keywords. The choice of keyword determines the variable's scope and mutability.

  • var: Function-scoped or globally scoped, can be re-declared and updated.
  • let: Block-scoped, can be updated but not re-declared in the same scope.
  • const: Block-scoped, cannot be updated or re-declared, used for constants.

Examples:

// Variable declaration and initialization
let name = 'John';
const age = 30;

console.log(name); // Output: John
console.log(age);  // Output: 30

// Trying to reassign a const variable will cause an error
// age = 31; // Uncaught TypeError: Assignment to constant variable

Data Types

JavaScript has several data types that can be categorized into primitives and objects:

  • Primitive Data Types:
    • String: Represents text. Example: 'hello'
    • Number: Represents numeric values. Example: 42
    • Boolean: Represents true or false
    • Null: Represents the absence of value. Example: null
    • Undefined: Represents a variable that has not been assigned a value. Example: undefined
    • Symbol: Represents a unique identifier. Example: Symbol('description')
    • BigInt: Represents integers with arbitrary precision. Example: 1234567890123456789012345678901234567890n
  • Objects: More complex data structures that include arrays, functions, and custom objects.

Examples:

let message = 'Hello, World!';
let count = 10;
let isActive = true;
let user = null;
let notDefined;
let uniqueSymbol = Symbol('id');
let largeNumber = 1234567890123456789012345678901234567890n;

console.log(typeof message); // Output: string
console.log(typeof count);   // Output: number
console.log(typeof isActive); // Output: boolean
console.log(typeof user);    // Output: object
console.log(typeof notDefined); // Output: undefined
console.log(typeof uniqueSymbol); // Output: symbol
console.log(typeof largeNumber); // Output: bigint

Functions

Functions in JavaScript are blocks of code designed to perform a particular task. They can be defined using function declarations or function expressions. Functions can also accept parameters and return values.

  • Function Declaration: Defined with the function keyword.
  • Function Expression: Defined as part of an expression, often used with anonymous functions.

Examples:

// Function Declaration
function greet(name) {
    return 'Hello, ' + name + '!';
}

console.log(greet('Alice')); // Output: Hello, Alice!

// Function Expression
const multiply = function(a, b) {
    return a * b;
};

console.log(multiply(5, 3)); // Output: 15

// Arrow Function
const add = (a, b) => a + b;

console.log(add(4, 6)); // Output: 10

Control Flow

Control flow statements manage the flow of execution in a program. They include conditionals and loops:

  • Conditionals: Used to execute code based on certain conditions. Examples include if, else if, and switch.
  • Loops: Used to repeat code multiple times. Examples include for, while, and do...while.

Examples:

// If-Else Statement
let score = 85;

if (score >= 90) {
    console.log('A');
} else if (score >= 80) {
    console.log('B');
} else {
    console.log('C');
}

// Switch Statement
let day = 2;
switch (day) {
    case 1:
        console.log('Monday');
        break;
    case 2:
        console.log('Tuesday');
        break;
    default:
        console.log('Other Day');
        break;
}

// For Loop
for (let i = 0; i < 5; i++) {
    console.log(i);
}

// While Loop
let count = 0;
while (count < 5) {
    console.log(count);
    count++;
}

// Do-While Loop
let number = 0;
do {
    console.log(number);
    number++;
} while (number < 5);

Objects

Objects are collections of key-value pairs. They can be used to store multiple values in a single variable. Each key is a string (or Symbol) and is associated with a value, which can be any data type.

Examples:

// Object Literal
const person = {
    name: 'John',
    age: 30,
    greet: function() {
        return 'Hello, ' + this.name;
    }
};

console.log(person.name); // Output: John
console.log(person.greet()); // Output: Hello, John

// Accessing Object Properties
console.log(person['age']); // Output: 30

// Adding Properties
person.email = 'john@example.com';
console.log(person.email); // Output: john@example.com

Arrays

Arrays are used to store multiple values in a single variable. They can hold values of any type and are indexed, meaning each value has a numeric position starting from 0.

Examples:

// Array Declaration
const fruits = ['apple', 'banana', 'cherry'];

console.log(fruits[0]); // Output: apple
console.log(fruits.length); // Output: 3

// Array Methods
fruits.push('date');
console.log(fruits); // Output: ['apple', 'banana', 'cherry', 'date']

fruits.pop();
console.log(fruits); // Output: ['apple', 'banana', 'cherry']

fruits.shift();
console.log(fruits); // Output: ['banana', 'cherry']

fruits.unshift('apricot');
console.log(fruits); // Output: ['apricot', 'banana', 'cherry']

// Iterating over an Array
fruits.forEach(function(fruit) {
    console.log(fruit);
});

Events

Events are actions that occur in the browser, such as user interactions (clicks, key presses) or other changes. JavaScript can handle these events to create interactive web applications.

Examples:

// Adding an Event Listener
document.querySelector('button').addEventListener('click', function() {
    alert('Button clicked!');
});

// Handling Form Submission
document.querySelector('form').addEventListener('submit', function(event) {
    event.preventDefault(); // Prevents the form from submitting the default way
    console.log('Form submitted');
});

Advanced Topics

Advanced JavaScript topics include more complex concepts and features:

  • Closures: Functions that have access to variables from their outer scope even after the outer function has finished executing.
  • Asynchronous JavaScript: Handling asynchronous operations using callbacks, promises, and async/await.
  • Prototypes: The mechanism by which JavaScript objects inherit features from one another.
  • ES6 Features: New syntax and features introduced in ECMAScript 6, such as arrow functions, template literals, and destructuring.

Examples:

// Closure Example
function createCounter() {
    let count = 0;
    return function() {
        count++;
        return count;
    };
}

const counter = createCounter();
console.log(counter()); // Output: 1
console.log(counter()); // Output: 2

// Promise Example
const fetchData = new Promise((resolve, reject) => {
    setTimeout(() => {
        resolve('Data fetched');
    }, 2000);
});

fetchData.then(data => {
    console.log(data); // Output: Data fetched
});

// Async/Await Example
async function fetchDataAsync() {
    let data = await fetchData;
    console.log(data); // Output: Data fetched
}

fetchDataAsync();
