Data Types and Operators

Understanding data types and operators is essential for programming in C. This section will cover the different data types available in C and the various operators that can be used to manipulate these data types.

Data Types

C has several basic data types, including:

  • int: Used to store integers (whole numbers).
  • float: Used to store floating-point numbers (numbers with decimal points).
  • double: Used to store double-precision floating-point numbers.
  • char: Used to store single characters.
// Example of data types
int myInt = 5;
float myFloat = 5.99;
double myDouble = 9.98;
char myChar = 'A';

Operators

Operators in C are used to perform operations on variables and values. C has a rich set of operators, including:

  • Arithmetic Operators: +, -, *, /, %
  • Assignment Operators: =, +=, -=, *=, /=, %=
  • Comparison Operators: ==, !=, >, <, >=, <=
  • Logical Operators: &&, ||, !
  • Bitwise Operators: &, |, ^, ~, <<, >>
// Example of operators
int a = 10;
int b = 20;
int sum = a + b;  // Arithmetic operator
int isEqual = (a == b);  // Comparison operator
int logicalAnd = (a > 5) && (b < 30);  // Logical operator
int bitwiseAnd = a & b;  // Bitwise operator
