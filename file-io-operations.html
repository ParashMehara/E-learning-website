File I/O Operations

File Input/Output (I/O) operations in C allow you to read from and write to files. This is essential for handling data storage and retrieval. This section covers the basics of file I/O operations.

Opening and Closing Files

Files are accessed using file pointers. The functions fopen and fclose are used to open and close files, respectively.

Opening a File

Use the fopen function to open a file. You can specify the mode for opening the file (e.g., read, write, append).

FILE *file_pointer;
file_pointer = fopen("filename.txt", "r"); // Open for reading
file_pointer = fopen("filename.txt", "w"); // Open for writing
file_pointer = fopen("filename.txt", "a"); // Open for appending

Closing a File

Use the fclose function to close an open file.

fclose(file_pointer);

Reading from and Writing to Files

After opening a file, you can read from or write to it using various functions.

Writing to a File

Use fprintf to write formatted text to a file and fwrite to write binary data.

#include <stdio.h>

int main() {
    FILE *file_pointer = fopen("output.txt", "w");
    if (file_pointer == NULL) {
        printf("Error opening file\n");
        return 1;
    }

    fprintf(file_pointer, "Hello, File I/O!\n");
    fclose(file_pointer);

    return 0;
}

Reading from a File

Use fscanf to read formatted text and fread to read binary data.

#include <stdio.h>

int main() {
    FILE *file_pointer = fopen("output.txt", "r");
    char buffer[100];

    if (file_pointer == NULL) {
        printf("Error opening file\n");
        return 1;
    }

    while (fgets(buffer, sizeof(buffer), file_pointer) != NULL) {
        printf("%s", buffer);
    }

    fclose(file_pointer);

    return 0;
}

Error Handling

It is important to handle errors when performing file operations. Check if the file pointer is NULL to ensure that the file was opened successfully.

Example

#include <stdio.h>

int main() {
    FILE *file_pointer = fopen("nonexistent.txt", "r");
    if (file_pointer == NULL) {
        perror("Error");
        return 1;
    }

    // File operations here

    fclose(file_pointer);

    return 0;
}
